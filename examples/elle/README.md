### ♡ **Why does this exist?**

* I had wanted to write a Raytracer for a while, and I finally got the opportunity to do so!

### ♡ **How to build?**

`make` (or `make run` to also run the raytracer)

(Note: you may have to set `ELLE_LIBS` to link with OpenGL for linux.)
